# 📧 Email Validator (Python)

A simple yet robust Python program designed to validate email inputs with clean logic and enhanced handling of edge cases.

---

## 🚀 Features

* ✅ Checks for empty input
* 🚫 Prevents spaces in email
* 📧 Ensures presence of `@` symbol
* 🔢 Validates that only one `@` exists
* 🧼 Uses input sanitization (`strip()`) for cleaner validation
* ⚡ Structured logic for better readability and accuracy

---

## 🛠️ Tech Stack

* Python (Core Concepts)
* Conditional Statements
* String Handling

---

## 📂 How It Works

The program takes user input and validates it step by step:

1. Removes unnecessary spaces using `.strip()`
2. Checks if input is empty
3. Ensures no spaces are present
4. Verifies the presence of `@`
5. Confirms only one `@` is used

---

## ▶️ Usage

Run the script:

```bash
python email_validator.py
```

Then enter your email when prompted.

Valid email
